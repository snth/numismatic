## Install

    pip install numismatic

## Run

Run the `coin` script from anywhere and explore from there

    coin

    coin list

    coin info

    coin info -a BTC,ETH,XMR,ZEC

    coin prices

    coin prices -a BTC,ETH,XMR,ZEC

    coin listen -a BTC,ETH,XMR,ZEC collect run
